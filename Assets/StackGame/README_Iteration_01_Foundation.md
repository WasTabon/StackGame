# README — Iteration 1: Фундамент

## Что изменилось
Первая итерация — создание базы проекта:
- **BlockLayer.cs** — процедурально генерирует меш блока (куб с 4 боковыми сторонами, каждая разделена на 2 цветные половины с тёмной разделительной полоской). Хранит индексы цветов, поддерживает вращение данных CW/CCW.
- **Tower.cs** — менеджер башни. Спавнит слои, управляет списком, задаёт случайные цвета.
- **GameColors.cs** — палитра из 5 ярких цветов на тёмно-синем фоне (красный, бирюзовый, фиолетовый, оранжевый, зелёный).
- **CameraController.cs** — камера под углом, смотрит на центр башни.
- **VertexColor.shader** — Surface shader с поддержкой vertex colors и Lambert освещением.
- **SetupIteration1.cs** — Editor скрипт для настройки сцены.

## Как настроить

1. Скопировать папку `Assets/STACK` в проект Unity.
2. Убедиться что DOTween Free установлен (пока не используется, но будет нужен).
3. Открыть сцену Gameplay (или создать пустую сцену).
4. В меню Unity: **STACK → Setup Gameplay Scene (Iteration 1)**.
5. Скрипт создаст: Tower с 6 слоями, камеру, освещение, материал.

## Как тестировать

- После запуска editor скрипта на сцене должна появиться башня из 6 цветных блоков.
- Камера смотрит на башню сбоку-сверху.
- Фон — тёмно-синий.
- Каждый блок — куб с 4 сторонами, каждая разделена вертикальной тёмной полоской на 2 цветные половины.
- Цвета случайные из палитры 5 цветов.
- Мягкое освещение с голубоватым ambient.

## Ожидаемый результат

Статичная сцена с башней из 6 разноцветных блоков на тёмно-синем фоне. Управление пока не реализовано — это чисто визуальный фундамент.
